# Fonts

NGIN#9 uses a small, intentional set of fonts.  
Each font has a specific role — do not substitute or mix usage casually.

Consistency is critical to maintaining the game's visual identity.

---

## Bebas Neue

**File:** BebasNeue-Regular.ttf  
**Source:** https://fonts.google.com/specimen/Bebas+Neue?preview.script=Latn  

Used for:
- Titles
- Section headers
- Large UI labels

Characteristics:
- Clean
- Condensed
- Highly readable at large sizes

> This is the primary display font of the game.

---

## Perfect DOS VGA 437

**Source:** https://www.dafont.com/perfect-dos-vga-437.font  

Used for:
- Terminals
- Computer-style interfaces
- Command output

Characteristics:
- Pixel-based
- Retro DOS aesthetic
- Low-level system feel

> Do NOT use for general UI. This font is reserved for system-style presentation.

---

## Saira Stencil One

**File:** SairaStencilOne-Regular.ttf  

Used for:
- Logo
- Industrial signage
- In-world markings

Characteristics:
- Bold
- Stenciled
- Industrial tone

> This font reinforces the mechanical and industrial identity of the world.

---

## Godot Default Font

Used for:
- General UI
- Body text
- Menus and standard interfaces

Characteristics:
- Highly readable
- Stable across resolutions
- Neutral appearance

> This is the backbone font. Use it wherever clarity is the priority.

---

## Summary

| Font                    | Primary Use                  |
|------------------------|------------------------------|
| Bebas Neue             | Titles, headers              |
| Perfect DOS VGA 437    | Terminals, system text       |
| Saira Stencil One      | Logo, industrial signage     |
| Godot Default Font     | General UI and body text     |

---

> Rule of thumb:  
> If it needs to be **read easily**, use the Godot default.  
> If it needs to **feel like a system**, use DOS.  
> If it needs to **feel industrial**, use stencil.  
> If it needs to **stand out**, use Bebas Neue.
